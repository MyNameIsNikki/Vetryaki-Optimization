## рҹ§  РҗР»РіРҫСҖРёСӮРјСӢ

### ACO (Ant Colony Optimization)

* РһРҝСӮРёРјРёР·Р°СҶРёСҸ РјР°СҖСҲСҖСғСӮР°
* РЈСҮРөСӮ СҖР°СҒСҒСӮРҫСҸРҪРёСҸ Рё СғРіР»РҫРІ
* РқР°СҒСӮСҖР°РёРІР°РөРјСӢРө РҝР°СҖР°РјРөСӮСҖСӢ (Оұ, ОІ, ПҒ)

### Genetic Algorithm

* РӯРІРҫР»СҺСҶРёРҫРҪРҪСӢР№ РҝРҫРҙС…РҫРҙ
* РҡСҖРҫСҒСҒРҫРІРөСҖ Рё РјСғСӮР°СҶРёРё
* РҹРҫРёСҒРә РіР»РҫРұР°Р»СҢРҪРҫРіРҫ РјРёРҪРёРјСғРјР°
